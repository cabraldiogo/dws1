HTML Element definitions

Chapter 2 elements definition

<h1> | <h2> | <h3> | <h4> | <h5> | <h6>
<h1>The Headings tags has 6 levels. The <h1> is the most important and has a bigger font size. The <h6> is the least important and has the smaller font size.</h1>
<p>
<p>The paragraph tag is used to wrap a text of a single letter, a word or many sentences.</p>
<b>
<b>The bold tag make any text appears bold when enclosed by this tag</b>
<sup>
The <sup> tag is used to put a text supscript. It is most used to dates suffixes. It also used to mathematical expression like power of a number.
<sub>
The <sub> tag is used to put a text subscript. It is most used for chemical formulas.
<br />
This tag is used to create a page break. It could be used inside a paragraph tag.
<hr />
The <hr /> tag create a break between sections drawing an horizontal line.
<strong>
The strong tag is used to give emphasis on a word or sentence. By default the browsers will show the text inside the these tags in bold style. The screen readers will read it with emphasis.
<blockquote>
This tag is used to wrap long quotes and the browsers indent its content.
<q>
This tag is used to wrap short and in-line quotes. The browsers put quotation marks at the beginning and the end of its content.
<abbr>
This tag is used for abbreviations. The full term must be set as a attribute on the opening tag. The full term is shown when the mouse is over the abbreviation.
<cite>
This tag is used to reference a book, film or any other piece of work. By default, browsers will show its content in italics.
<dfn>
This tag is used when you need to write a new or technical terminology for the first time, and it is used to define the term.
<address>
This tag is used to wrap the page's author contact details. Most browsers will show it in italics

<ins>
This tag is used to show content added to the document.It is shown underlined
<del>
This tag is used to show content that was deleted from the document. It is shown with line through it
<s>
This tag is used to show content is not accurate anymore, but should not be deleted. It is shown with line through it

Chapter 3 elements definition

<ol> and <li>
The <ol> stands for Ordered List. Each element of the list should be written using the <li> tag that stands for List Item. By default the browsers indent its elements and put then in numeric order
<ul>
The <ul> stands for Unordered List. Each element of the list should be written using the <li> tag that stands for List Item. By default the browsers indent its elements and uses circles, squares or diamonds in front of any list item.
<dl> | <dt> | <dd>
The <dl> tag stands for Definition List. It contains a series of terms and definitions.
Each term that will be defined should be wrapped using the <dt>, that stands for Definition Term. The definitions itself should be wrapped using the <dd> tag that stands for Definition.
