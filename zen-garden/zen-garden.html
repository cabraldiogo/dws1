Zen Garden Assignment

Instructions

  1. Go to CSS Zen Design website and choose three different designs.
  2. Explore their CSS styling using Inspect Element in your web browser and selecting "View This Design's CSS" under Resources.
  3. Explain why you like these three designs and how the designs utilize CSS to accomplish different presentations while using exactly the same HTML and content

A Robot Named Jimmy by meltmedia

  • I really like the way they used a z-index that makes the robot to be covered when we scroll down the page.
  • I like the color choice and simplicity of elements.
  • They used a lot of float, padding, margin and borders attributes on the CSS code.
  • Responsive design.
  • The CSS validator shows 12 erros related to animation.

Oceanscape by Justin gray

  • I like the way the ocean gets darker when we scroll down the page.
  • I like the simplicity of colors and font choice. However I think they should have used a bigger font-size.
  • They used a some attributes that I didn't understand such as "white-space" and "overflow".
  • The CSS code passed with no erros at the W3C validator.

Leggo My Ego by Jon Tan

  • I like the design style inspired by Lego pieces.
  • They used images as the h3 elements.
  • They used a three-colunm layout.
  • They used a lot of letter-spacing and word-spacing.
  • The CSS code passed with no erros at the W3C validator.
